1.1 ¿Tienes 15 minutos y quieres aprender a usar Git?

Git permite a grupos de programadores trabajar sobre los mismos documentos (generalmente código) de forma concurrente, y sin ir molestándose los unos a los otros. Es un sistema de control de versiones.

El prompt del terminal está situado en el directorio raíz. Para inicializar el repositorio de Git aquí, escribe el siguiente comando:

git init

1.1 Got 15 minutes and want to learn Git?

Git allows groups of people to work on the same documents (often code) at the same time, and without stepping on each other's toes. It's a distributed version control system.
