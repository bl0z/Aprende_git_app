1.10 Repositorios remotos

Vamos a intentar conectarnos a un repositorio que hemos creado en GitHub para hacer algunas pruebas. Para subir tu repositorio local a los servidores de GitHub, lo primero que tenemos que hacer es añadir el repositorio remoto.

El comando recibe el nombre del repositorio local y la dirección web del repositorio remoto, en este caso, https://github.com/test/test.git.

Prueba a escribir el siguiente comando y no olvides incluir los diferentes parámetros:

git remote add origin https://github.com/test/test.git

1.10 Remote Repositories

Great job! We've gone ahead and created a new empty GitHub repository for you to use with Try Git at https://github.com/test/test.git. To push our local repo to the GitHub server we'll need to add a remote repository.

This command takes a remote name and a repository URL, which in your case is https://github.com/test/test.git.

Go ahead and run git remote add with the options below:

git remote add origin https://github.com/test/test.git
