1.13 Removing All The Things

Vale, los amigos que invitamos han hecho un buen destrozo en nuestro proyecto y queremos borrar todos los ficheros para empezar de nuevo este proyecto. Puedes usar el comando git rm para borrar no solo los ficheros del repositorio local, sino que también los del servidor remoto.

Nuevamente, puedes usar el comodín para hacerte el trabajo más fácil:

git rm '*.txt'
