1.4 Añadiendo cambios

Bien hecho, parece que nuestro repositorio Git funciona correctamente. ¿Te has fijado en como Git nos dice que holamundo.txt es un nuevo fichero? Eso significa que Git comprueba y nos informa de que holamundo.txt es un fichero nuevo.

Para decirle a Git que queremos empezar a hacerle un seguimiento al fichero holamundo.txt, lo primero que tenemos que hacerlo es añadirlo a la zona de pruebas usando git add.

git add 'holamundo.txt'
